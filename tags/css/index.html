nginx 处理带版本号的css文件expires

  • css
  • expires
  • nginx
  • 版本

例如：http://bdwm.info/wp-content/themes/minimatica/style.css?ver=1.0.1

在location / {}内加入:
if ($request_uri ~* ".(ico|css|js|gif|jpe?g|png)\?[0-9]+$") {
expires max;
break;
}
