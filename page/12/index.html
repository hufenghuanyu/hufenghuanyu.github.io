nginx:413 Request Entity Too Large

2011-10-31
Ubuntu 10.04 下用nginx官方源安装的nginx，上传文件时出现413 Request Entity Too Large，查了一下发现client_max_body_size这个参数限制了上传文件的大小，默认是1M,需要配置一下上传文件最大值: 1.打开nginx主配置文件sudo vim /etc/nginx/nginx.conf 2.在http模块段添加client_max_body_size 20m（改为20m） 3.刷新nginx服务sudo /etc/init.d/nginx restart
