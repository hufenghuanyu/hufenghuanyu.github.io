acer As3641装Mac OS 10.4.8成功

2007-02-21 | 未分类 | 1.1k 字 | 3 分钟
文章目录

　　最近在家里无聊又想装苹果漂亮的系统.一方面,vista比较让人失望,比漂亮比不过mac os,比兼容性不如xp,在先后装了两次之后,还是放弃了,至少暂时放弃.去年十月份的时候折腾过一段时间mac os,当时装的是mac os 10.4.3的ghost版,那个版本安装比较简单,跟windows的ghost操作一样,当时装好之后,网卡直接能用,声卡显卡统统不行,先后装过好多次,只是显卡驱动装不上,不能调整分辨率和刷新率,默认的1024x768,看着很不爽,很模糊,无奈还是格了:(
　　对vista比较失望，于是试着去baidu了一下ATIx200M的显卡安装Mac OS的情况，好像是可以装了，于是开始下载，准备装。Windows XP和Mac OS双系统比较完美：）这次还是想装ghost版的，有经验而且相对比较容易。不过下载的时候遇到了困难，10.4.6版比较完美的ghost版,我以平均几十k的速度下了两天下到99%,怎么也没速度了,于是只好转而开始下载最新的非ghost版的,这个到比较顺利,速度也比较快,很快下完,然后就开始安装.刚开始随便找了一个教程用虚拟机vm安装,为了避免操作失误损坏数据的现象发生,我先用的移动硬盘安装,遇到找不到磁盘的情况,查了一下发现是磁盘问题,之前也没有人用移动硬盘安装成功的例子,于是放弃移动硬盘安装.开始格盘重新划分出来新的分区装.按照之前找到的教程装,这次比较顺利，很快装完重启.结果启动不起来,出现”chain0 boot error”,换了一个版本的chain0和com.apple.Boot.plist,还是不行,出现HFS System错误,我分析了一下,这次失败的主要原因应该是当时分区的时候新分到的盘位于disk(0,2)的位置,也就是第三个磁盘,对于chain0来说，只能搜索前8.4G的启动信息;而且虚拟机装完没提示重新启动我就关了虚拟机的电源,这两点可能是主要原因.心有不甘,于是卷土重来,重新调整磁盘分区,把C盘调整为8G,然后紧跟着7G的Mac分区;而且找了一个相当详尽的教程,这个教程里面说的机器跟我的机器一样,都是ATI Xpress 200的芯片组,按照教程一点点小心翼翼的安装,基本安装过程比较顺利,虚拟机安装完成之后重启,终于进入了久违的白白的苹果界面,这个时候跟上次一样,还是默认的1024x768分辨率,没有声音.开始折腾显卡驱动,刚开始比较受打击,搜到的关于我这样芯片组和集成显卡的机器都搞不定显卡的问题,官方驱动不可能,只好找第三方驱动,之前我装的是omni的Callisto的b003版的驱动,装好之后,重启可以调整分辨率和刷新率,但是调成我的宽屏分辨率之后,屏幕狂闪不已.这次下载到b003和b005俩版本的驱动,直接去试005的,安装好重启,试着一下的心态进入系统调整,发现可以调分辨率,而且不闪,非常开心:)信心大增,再接再厉开始找声卡驱动,还好我的声卡比较大众化,比较容易解决,试了第二个就可以了,不过只能出声,不能用Mic,还好有声音了,开心^_^
　　现在剩下的问题包括,显卡没有开QE/CI,播放rm/rmvb的视频有点卡;声音不能录入;无线网卡找不到驱动
　　这次要是搞定无线网卡一般使用就基本没问题了,问题和体会:建立虚拟机的时候要选择整个磁盘,不能选择单个分区,否则安装的时候找不到磁盘.这个问题那位大虾也提到了,按照他的提示去做,基本不会遇到什么问题.enjoy Apple,enjoy Mac OS :)
