ATI显卡 for Vista 移动版驱动作方法 zz

自己的9600战斗版没有for Vista的驱动，而M$自带的驱动又用的很是不爽，最后总算用DH Mod自己做了for Vista的驱动，把方法写下来，也算是对和我一样为找不到驱动而苦恼的人做点贡献。

很多有本的人用的都是配了ATI独显的本，还有好多台式机用的也是ATI移动版、战斗版的显卡（比如 Mobility9600），但是用这些显卡装Vista的话还有一个很大的问题，ATI没有发布Mobility系列for Vista的驱动。虽然勉强可以用Vista自带的驱动，可是应该谁都想自己的显卡表现更好一点吧，Viats自带的驱动显然无法满足要求。

可以用DH Mobility Modder.NET来制作属于自己的for Vista驱动，对ATI独显本本熟悉的人应该知道这个软件，一直以来许多笔记本用户都使用DH Mod这个终极武器，通过几个简单的步骤，DIY出支持自己移动显示芯片的最新版催化剂驱动。

第一步 （64位系统自行下载相关驱动，地址不再给出）

下载 DH Mobility Modder.NET 0.8.7.0 Beta版For WinXP/Vista-32，地址：http://drivers.mydrivers.com/drivers/182-73126

下载 ATI Radeon 9550/9600/9700/9800/X300/X600/X700/X800/X1300/X1600/X1800/X1900/X1950/HD 2400/HD 2600/HD 2900 XT系列显卡最新催化剂驱动7.8官方正式完整(控制中心)多语言版For Vista-32，地址：http://drivers.mydrivers.com/drivers/181-72654-ATI-Radeon-X1900-X1950-HD-2400-HD-2600-HD-2900-XT-7.8-(-)-For-Vista-32/

下载 Microsoft.NET Framework 2.0（已经安装过的此步可省略），地址：http://drivers.mydrivers.com/drivers/135-54350

下载 MSXML 4.0（已经安装过的此步可省略），地址：http://www.newhua.com/soft/16288.htm

第二步

1.没有安装 Microsoft.NET Framework 2.0 和 MSXML 4.0 的先安装这两个组件，已经安装的跳过此步

2.将下到的驱动解压到任意路径下（不要安装），打开下到的DH Mobility Modder.NET目录中的MobilityDotNET.exe，选择驱动所解压到的目录后点击Modify，等待完成后推出。找到驱动所解压的目录，执行setup安装即可。

注：此方法适用于没有安装显卡驱动的情况，如果已经安装过较早版本的驱动，需要安装新版本驱动，则先将较早版本驱动卸载后在按如上方法操作

吼吼，写完了，装上驱动感觉就是好，这个方法同样适用于xp，不过xp下面是有驱动的，不建议这样搞，当然想尝试的也可以试试。
