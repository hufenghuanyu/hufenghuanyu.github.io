1、MSCD 龙帝国技术社区

谈到龙帝国相信大家肯定不会陌生,脑海肯定浮现出N多系统方面的高手,它是一个研发能力非常强的论坛,拥有系统界众多高手,比如死性不改,陈辉,西门大哥, fxzm,研究僧人,木鸟,风之客,匪兵.

龙帝国可以说是GHOST封装技术的鼻祖,素有封装技术首发之地,到现在为止,很多版本的系统都在延用龙帝国的封装工具.

代表作品:

死性不改的S&R&S9.7封装工具, GHOST XP SP2 OEM 21合1 版;

陈辉的DLLCACHEMANAGER V2.0

西门大哥的GHOST XP SP2电脑公司专业版(与东海7.0没关系)V7.0

FXZM(他非常受网友的肯定,不过现在归隐山林啦) GHOST XP1008版本,10008更新版0425更新版

研究僧人的研究僧人Ghost_XP_1109 MSCD龙帝国2.0极度纯净版

木鸟的万能驱动包

风之客的GHOST XP 5.0 BY风之客

匪兵的匪兵工作室电脑城极速装机版V3.0

2、龙卷风极品论坛

可能很多人了解龙卷风都是从它的AD区开始的,但龙卷风的系统研发方面绝对可以称得上一流,DOZ大侠的各各版本,都已经流传为经典.DOZ的系统以绝对纯净著称,一键在光启界面修改DMI激活OEM堪称首创.到现在为止还有很多人在用他的作品.但DOZ大侠自从发布2.7以后就说,XP不出SP3就不会再出系统.嘿嘿,这句话很耐人寻味,很吊人胃口

代表作品: Windows.XP.SP2.龙卷风免激活版V2.7

           Windows.2003.SP1.龙卷风大客户版V2.5

           Ghost.Windows.XP.SP2.龙卷风纯净版V1.0

关键词 “真纯净”

3 、番茄花园

如果说前面两个论坛是以技术著称的话,那么番茄的系统就是美化安装系统的代表啦,坛主大番茄以美化这一词打动N多盗版商的心,在全国各大电脑城盗版小摊里,我们都可以看到番茄系统的身影,但随着版本的升级,系统渐渐地加入了很多恶意软件,比如令人唾指的雅虎助手与网络实名,让很多番茄迷甚似伤心,人气大不如前,再加之现在GHOST XP 系统的技术越来越纯熟,安装版的人气就变得更加大不前.总之如果你需要美化的话,请试试番茄的系统吧

代表作品: 番茄花园 Windows XP Pro SP2 免激活 V 3.1

           番茄花园 GHOST XP SP2 v 2.6

           番茄花园 Windows XP SP2 美化版 V5.1(BY SKY)

关键字 “美”

4、东海电脑公司论坛

东海的系统以GHOST系统为著长,其系统专为电脑城快速装机而研发的,号称15分钟装机打包带走,其特点以稳定,快速,驱动包全.曾几何时,拥有极多FANS,从当初的5.0到6.0以及别人修改的6.1一直到6.6都是备受大家的青睐, 7.0首创驱动包可选择性安装和提示删除驱动包,可见其技术很强.但7.0上市以来,由于捆绑网络实名与雅虎助手,人气大不如前,导致很多论坛打出”本论坛不许讨论东海7.0的声明”可见其被讨论激烈性与普遍性,但最近东海相继发布更新版7.2 7.3逐步删掉网络实名与雅虎助手,但还是不够绝对纯净..但人气已提高不少.

代表作品: 东海电脑公司GHOST XP 5.0,6.0,7.0

关键字 “快”

5 、JUJU猫宽带宝藏论坛

龙帝国众位高手原本在JUJU猫论坛,后来因为有分歧就此分道扬镳了,从而创立今天的龙帝国,可见JUJU猫资格之老,谈其系统,素以GHOST系统见长,论坛中有以杨海军的软件安装管理器和jujumao封装工具2.0为代表的优秀原创作品,但由于系统不够纯净以及其它系统的分流,人气已经大不如前,而且由于论坛版面广告过多更加混乱,比之以前更加萧条.自从VISTA上市,JUJU猫专攻美化这一奇招,收到市场不错的好评

代表作品: JUJUMAO Winxp SP2电脑城装机8.2纯净(VISTA模仿版)

JUJUMAO网吧专用版WINXPSP1

关键词 “曾经”

6、 系统之家

系统之家可以称得上是后起之秀,在上次全国BBS评选中,以系统集粹这版块打入全国前500名BBS,可见其人气之高,子版系统评测可以说是菜鸟的天堂,给广大网友提供最新系统的评测. 站长幸运星,主攻GHOST系统,其作品以纯净快速装机为特色,吸引不少系统高手加盟,如 Tanison 5x54 (VISTA精简版的作者) 姗姗来迟,但去年受到黑客攻击网站曾一度关闭,损失了不少人气,系统评测区的评测高手走的不少,这一特色版块变得大不如前,回帖人数寥寥无几.由于幸运星时间不够充裕也导致作品更新过慢.导致曝光率不够,盗版市场很少见到此系统.但新年伊始,管理T队对论坛进行整体改革,论坛人气有所回升.

代表作品: 系统之家 Win XP 快速装机光盘0610,0612

第六版Win98/第三版Win2000 二合一

系统之家老机专用Home XP Sp1a精简版

关键词 “后起之秀”

7 、我们的深度论坛

如果以上这些论坛是以主攻WINXP GHOST以及安装版本的话,那么深度论坛就以主攻WINXP精简系统,其系统素以精简著名,在各各精简系统中,无人出其右,站长Deepin更是在其论坛与作品这中不加任何广告,而受到很多网友认同,现在已拥有不少铁杆FANS,去年直接把水区关掉,可见专攻技术的决心.论坛中也不乏美化高手,掸子大侠,WINDOWS美化主题包也可以说是一绝.WIN2003高手冷风,也曾常发布最新作品.但个人认为,由于PC行业硬件不断地拓展,硬件对于系统的束缚也越来越少,我敢断言XP GHOST系统与VISTA精简系统才是发展潮流,随着电脑配置的增强，硬件的不断更新XP精简系统早晚有一天会退出历史的舞台,深度又将何去何从,站长如何发展我们的论坛，我们拭目以待.

代表作品: Deepin XP Lite V5.6

Deepin XP SP2 Plus 增强优化版

Win2K3_EE_0218_2IN1精简版

掸子的N多美化主题

关键词 “何去何从”

8 、雨林木风

该论坛致力于原创,对各家系统广泛收集,在系统界很有信誉,站长罗文更是管理人性化,使该论坛聚集着不少系统高手,诸如X-TIGER Xiaodong 寒星等高手.其作品有很多人追捧,备受广大装机员的青睐,NTFS纯净系统更是首创. 而且各各系统都有新作,最近在专攻WINDOWS XPVISTA美化版.个人感觉,该系统美化包非常不错,可与番茄有一拼,在封装软件选择方面也非常合理.上升势头非常良好.是只潜力股

代表作品:

雨林木风GHOST XP 3.0 3.1

雨林木风GHOST XP VISTA美化纯净版

雨林木风WINXP SP2 EN版和(繁体版)

雨林木风ONE KEY GHOST11.0

雨林木风Ghost Win2003 SP1 R2企业版

关键字 “全”

9 、FLY飞杨社区

该论坛是属于刚刚起步,去年下半年才开,最近人气暴涨,这与其系统被网友所肯定是分不开的,现在该论坛越做越细,分出了好多系统研发的区,已经形成产业一条龙,坛主茶壶管理人性化,虚心采纳各方意见,系统有了质得飞跃,但不善于做GHOST封装系统,最近吸引不少系统封装高手,已有新盘发出,最近也在转型,主攻美化版系统,装机无忧系也是受到好评.最近由武毅慧和逍遥茶正在制作下一个新版GHOST, 论坛内部正在酝酿着一场革命,期待其有一个崭新的起点.嘿嘿AD一下,该论坛进行网上光盘教学,在会员中培养系统高手,以后的力量不可估量.

代表作品:WINXP 装机无忧1.0 1.

FLY社区 Windows XP PRO SP2 免激活正式版

WindowsXP双模光盘（飞龙行天下－飞扬专用版）(由武毅慧 (wyh168 )和逍遥茶合做)

windows XP SP2免激活（建院版)专给学校定做的内部版

关键词 “无限的未来”

10、 新垦家园

新兴的一个系统创作论坛,比较有活力,管理员比较有创造力.清风明月,子涵均有自己的作品,也有不少系统高手加盟,比如 叶,匪兵, dreambrain.但由于系统版本众多,没有打造统一品牌,使该论坛在系统界影响不是很深.期待其有不错的系统出品.

代表作品: 新垦家园GHOST XP电脑城版 Bydreambrain

GHOST Windows XP-SP2 叶v3.3 正式版

Windows xp sp2 pro 纯净系统恢复盘 BY子涵

关键词 “期待”

11、999宝藏网

  

该论坛有着各种版本的系统,有点百家争鸣的味道,其宗旨是致力于系统光盘研究、发布,其中高手不少,有逆风飞扬 亮 金水桥 尼古拉斯他们的精简版都是一流的,也有世纪动力和sccsk等封装高手,特别是亮版的精简系统,有很好的口碑,也有不少的FANS,与DEEPIN的精简系统有得拼.该论坛现在开始转型专攻VISTA美化版,最近也会有新版本问市.必将吸引不少系统迷的关注.

代表作品: 999宝藏网 Windows xp 新年快乐版

武毅慧 (wyh168 )Mini xp sp2 万能Ghost 精简通用版

亮版Windows XP Pro SP1 精简版 V1.0

12、远景

不知道大家还记得一牛人花超千大洋买的VISTA旗舰版在远景首发挂种,一张张截图让大家切身的感受到VISTA来临的气息,此举让远景赚足了人气,注册用户爆长,看来该站长是着眼于未来,也是一只潜力股.

代表作品:VISTA系列

关键词 “未来”

我对系统界的发展史分为四个阶段

混沌期:

遥想当年,一张上海政府VOL或者一张电脑疯子MSDN盘,就可以装机无忧,大家肯定有过手动装机1个多小时并且无驱动包无软件,有点欲哭无泪的感觉,所有东西都需要自己一点点做,真是让人苦不堪言. 这时任何技术都处于萌芽期,都只是高手脑海里的一个 “X”. 石即将破,天将会惊..

三国鼎力:

随机各位系统高手们互相交流,技术逐渐成熟,这时番茄,东海,JUJU猫突出群围,一名惊人,自此三分盗版系统天下,惊似当年的三国演义. 这三个系统可是让那些盗版商赚得不亦乐乎,5块,就5块,不知道中国卖出了多少个5块.我的评语: “一句话,存在即合理”,也可以看出当初这三个系统满足市场需求,才有今天这样的成就.

群雄逐鹿:

当发现小小一张盘,有如此大的利润,各位潜水的高手,终于浮出水面, 当发现系统封装与安装已不再是那么神秘,众多高手开始敢于挑战权威,仔细钻研nlite与Easyboot以及自己的封装工具和驱动, 专心研究前者的系统,对其进行改良,着手于细节与美化,使自己的系统更人性化,至此中国系统界就变得有群雄逐鹿,决战于中原的味道.. 谁主沉浮,我们拭目以待!

未来时代:

谈到未来我们不得不说VISTA,就连联想的高级工程师都说,如果想流畅地跑VISTA最起码得4G内存,其它得就别说啦.所以很多人对VISTA非常不看好,前景不乐观,但不知大家还记得当年WINXP上市的时候,又有多少人反对它呢 说它过于臃肿,华而不实,漏洞百出,但XP比WIN2K漂亮,容易操作,最终为大家所认同.那么今天的VISTA是不是和当年的WINXP又有着惊人的相似呢 谁敢说技术就停滞不前了

所以,各有所长,各有所短,各位网友能找到最适合自己的系统才是最好的

本人从事电脑行业4年，这篇文章写了1个来小时，仅是我自己的看法，有些不成熟的地方还希望跟大家交流
